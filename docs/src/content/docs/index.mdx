OpenCheckout is an open-source, self-hosted checkout system for accepting payments on the web. It provides a hosted checkout page and a Checkout Sessions REST API that you deploy on your own infrastructure. Your customers pay using their [Open Payments](https://openpayments.dev) wallet address.
